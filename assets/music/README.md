# Royalty-Free Background Music Library

To use background music in ClipMind, place properly licensed royalty-free audio tracks (e.g., in `.mp3`, `.wav`, `.m4a`, `.ogg`, or `.flac` formats) into the emotion directories within this folder:

- `assets/music/uplifting/`
- `assets/music/inspirational/`
- `assets/music/calm/`
- `assets/music/dramatic/`
- `assets/music/emotional/`
- `assets/music/corporate/`
- `assets/music/default/` (guaranteed fallback)

## Companion License Files
To track licenses for copyright claims, you can place a JSON companion file with the exact same name as the music file plus `.json` (e.g. `track1.mp3` and `track1.mp3.json` or `track1.json`).

Format of the companion json file:
```json
{
  "license": "Creative Commons Attribution 4.0",
  "license_id": "CC-BY-4.0-12345"
}
```
